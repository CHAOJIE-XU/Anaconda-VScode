# Anaconda-VScode
這是用來記錄如何將anaconda的jupyer套在VScode環境中

jupyer裡面會有Tensorflow、numpy、其餘等等......的函式庫

希望這篇能幫助需樣在VScode中使用深度學習(DL，Deep learning)的人

內容僅供參考

## 使用版本
* python = 3.6.10
* Anaconda = 2.5.2
* tensorflow = 2.0.0
* numpy = 1.19.5

>未來有機會再更新
